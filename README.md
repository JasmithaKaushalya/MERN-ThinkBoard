# MERN ThinkBoard

A full-stack notes application built using the MERN stack, featuring a responsive UI and custom backend rate limiting. Completed as part of my development journey following the freeCodeCamp modern MERN tutorial.

## 🚀 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Security:** Upstash Redis (Rate Limiting)

## 📂 Project Structure
- `/backend` - REST API built with Express and MongoDB
- `/frontend` - Single Page Application built with React and Tailwind
